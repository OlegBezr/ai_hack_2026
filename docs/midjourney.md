Add the MCP server to your client:

   Claude Code:

   claude mcp add midjourney --transport http https://mcp.midjourney.com/mcp





   Claude Desktop / Cursor / other MCP clients:  Add to your MCP config:

   {

     "mcpServers": {

       "midjourney": {

         "url": "https://mcp.midjourney.com/mcp"

       }

     }

   }
